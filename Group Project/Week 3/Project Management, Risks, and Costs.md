#### Software Development Methodologies

requirements checks

- validity - have assumptions been made about the typical user
- consistency - check for conflicting requirements or constraints
- completeness - are all function and constraints covered
- realism - can system be built within budget and schedule
- verifiability - can conformance be proven to consumer
