**foldr**
(a -> b -> b) -> b -> [a] -> b

foldr (+) 0 [1, 2, 3] -- 1 + (2 + (3 + 0)) = 6

"recursively reduces a list from the right to a single value, using a combining function" 

**foldl**
(b -> a -> b) -> b -> [a] -> b

foldl (+)  0  [1, 2, 3] -- ((0 + 1) + 2) + 3 = 6

"it recursively reduces a list from the left to a single value, using a combining function"
