

**Different types of evaluation**

FORMATIVE - measure

SUMMATIVE - assess


Expert Evaluation

**experts and specialists find problems based on their experiences**

Heuristic Evaluation

**one or more evaluators inspect the UI against a list of recognized principles ('heuristics')**

Cognitive Evaluation

expert adopting the perspective of a first-time user

Using models

 Descriptive models
• For example, models of the task structure
• For analysis of effort (steps required) and complexity (depth of
goal structures)
• Predictive models
• For example, Fitts’ Law
• For example, KLM (Keystroke-level Model)

**What data can an evaluation generate?**

- Qualitative data
	- identification of usability issues
	- description of experience
	- observation of different behaviors

- Quantitative data
	- Performance measures: time, error rate
	- self-reported metrics
	- physiological data

- **Objective data**
	- Collected through observation
	- measurements
	- data logs

- **Subjective data**
	- Collected through communication
	- interviews, questionnaires
