### Supervised Learning


supervised learning algorithm is trained using labelled datasets
	each input is paired with a corresponding correct output
learn to map an input to a particular output
learn from experience to make predictions on new unseen data
similar to a teacher-student scenario

- the algorithm will teach itself to identify changes in Y based on the values of X
	- prediction, presented with new X, the algorithm will be able to predict how Y would be like
	- a dataset of labelled data is required
- The key characteristic of supervised learning: the learning algorithm is trained using examples that include the correct answers
- if Y is a numerical value, we call it a regression problem
- if Y is a category, we call it a classification problem


#### Regression

### Evaluation: performance metrics

- mean squared error (MSE) 
	- the average of the squared differences between predicted and actual values
	- it tends to emphasize larger errors due to squaring (focus on larger deviations)
- Mean absolute error (MAE)
	- The average of the absolute differences between predicted an actual values
	- it treats all errors equally, making it more robust to extreme values


### Regression and classification

- regression problem:

	- the output is a numerical value
- classification problem:

	- the output is a category
