1. What is authorization?

checks whether an access request for an object originating from a principal should be granted or denied
		reference monitor, Access control list, finding the rules applicable for a given access request

2. What is accountability?

mechanism to prove that someone performed a computer activity at a specific point in time
		audit checks etc

3. What is the difference between accountability and responsibility?

accountability is an attempt at proving someone performed a computer activity, whilst responsibility is just judging whether or not someone should be granted access to an object depending on who they are   

4. What are the Primary Security Concerns?

The PSC-CIA triad

- Loss of data **Confidentiality** 
- Loss of data **Integrity**
- Loss of data **Availability**  

5. Can you describe the Access Control Procedures

- access control procedures, the procedures which access is controlled between parties interested in the gaining and utilization of the things gained by the access

allocation of privileges - limit on the type of access
administration of privileges - recording/revocation of privileges
monitoring access
prevention of unauthorized access
identification and authorization


6. What is Mandatory Access Control and Discretionary Access Control

Mandatory Access Control

7. What is Access Control List

a list / file of users who are given the privilege of access to a system of resources

8. What is Role-Based Access Control

least privilege, the objective is to give people the least amount of access to a system that is needed to perform the job they're doing

role based is simplifying least privilege, by assigning users to a group, then assigning rights to the groups


9. Please describe Administration of Privileges

Handing over privileges
informing user of responsibilities associated with the privileges
recording and revocation of these administered privileges


10. Why monitor a system?


a defense against control loopholes

provides evidence of a security incident 

provides a model for normal behavior 

a user may seek to extend/escalate their privileges illegally, so evidence for these sorts of security incidents would need to be collected. its also useful to know the normal behavior of a system
