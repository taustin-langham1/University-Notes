1. **What is authentication and the difference from identification?**  

authentication is verification of identity
identification is determining who the user is 

2. **How many authentication factors we can use to design an authentication scheme?**  

3, a) what the user knows 2) what the user has 3) what the user is

4. **How many common methods to crack a textual password?**  

brute force dictionary

5. **What is graphical password?**  

users interact with images, rather than a text based approach

6. Can you name at least 5 types of biometrics?  

voice recognition, face recognition, finger print, signature recognition, iris recognition

7. What are the advantages of using biometric authentication?  

convenience, and resistant to phishing

8. What is multifactor authentication?

more than one factor of verification is required for the user to be authenticated
