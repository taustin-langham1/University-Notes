
##### Lists and zips - a focus on list comprehension, to construct and transform lists



basic syntax of a list:

[output expression | generator, conditions]


| = separates the output from generators/filters

generators (x <- list) pull values from a list

Conditions (even x) filter elements


**simple examples**

[x * 2 | x <- [1, 2, 3]] -- output [2, 4, 6]

with filtering

[x | x <- [1..5], even x] -- output [2, 4]


multiple generators

[(x, y) | x <- [1, 2], y <- ['a', 'b']] -- output [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]

**worked example**

[x^2 | x <- [1..5], x `mod` 2 /= 0] -- output [ 9, 25]
